# DAS Development Freelance Website

See it [live](https://das-next-js.vercel.app/)

## How to use

Download the example [or clone the repo](https://github.com/onTheDL/das-nextJS.git):

Install it and run:

```sh
npm install
npm run dev
```

## Stack

- JS, React.js
- Material UI, Html, CSS
- Firebase
- Next.js
